## 👋 欢迎

你好！欢迎来到我的个人博客。

这个博客使用 [Hexo](https://hexo.io/) 静态站点生成器构建，托管在 GitHub Pages 上。我将在这里记录技术学习笔记、项目经验和生活感悟。

## 🛠️ 博客技术栈

- **框架**：Hexo 7.x
- **主题**：自定义 Flavor 主题
- **托管**：GitHub Pages
- **评论**：Utterances（基于 GitHub Issues）
- **音乐**：网易云音乐外链播放器

## 📝 内容计划

我计划在这里分享以下内容：

1. **前端开发** - JavaScript、Vue、React 等
2. **后端技术** - Node.js、Python、数据库
3. **开发工具** - Git、Docker、VS Code 技巧
4. **项目实战** - 完整项目的开发过程
5. **学习笔记** - 阅读和课程笔记

## 🎯 写在最后
